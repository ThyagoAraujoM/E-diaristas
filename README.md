## 🧪 Tecnologias

Esse projeto foi desenvolvido com as seguintes tecnologias:

- [React](https://reactjs.org)
- [React-Native](https://reactnative.dev)
- [TypeScript](https://www.typescriptlang.org/)
- [Sass](https://sass-lang.com)
- [Next](https://nextjs.org)

## 💻 Projeto

E-diaristas é um projeto simples com react e react native com uma página que consume uma api. O uso da aplicação é para procurar diaristas disponíveis na região, totalmente ilustrativo e somente o cep 01.001.000 funciona.
